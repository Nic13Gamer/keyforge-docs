---
title: Delete license
---

You can delete a license using the SDK. Here is an example of how to delete a license:

```js
import { Keyforge } from 'keyforge-js';

const keyforge = new Keyforge('sk_1234');

await keyforge.licenses.delete('ABCDE-ABCDE-ABCDE-ABCDE-ABCDE');
```

## Parameters

<TypeTable
  type={{
    key: {
      description: 'The key of the license to get deleted.',
      type: 'string',
    },
  }}
/>
